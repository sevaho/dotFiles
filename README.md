#Dotfiles

&nbsp;

Install via wget

```bash
sh -c "$(wget https://raw.githubusercontent.com/sevaho/dotFiles/master/install.sh -O -)"
```

Install via curl

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/sevaho/dotFiles/master/install.sh)"
```

- Roboto
- AIcons.ttf
- fontawesome-webfont.ttf
